# 📊 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes the **Superstore Sales Dataset** to evaluate business performance and generate actionable insights.
The analysis focuses on revenue trends, product performance, and profitability to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze **monthly revenue trends**
* Identify **top-selling products**
* Perform **profitability analysis**
* Generate **business insights**

---

## 📂 Dataset Description

The dataset contains retail transaction data with the following key features:

* **Order Date** – Date of purchase
* **Sales** – Revenue generated
* **Profit** – Profit earned per transaction
* **Category / Sub-Category** – Product classification
* **Product Name** – Individual product details
* **Region** – Sales region

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas (Data Analysis)
* Matplotlib & Seaborn (Visualization)
* Jupyter Notebook

---

## 📊 Analysis Performed

### 1. Monthly Revenue Trends

* Aggregated sales by month
* Identified seasonal patterns and trends

### 2. Top-Selling Products

* Ranked products based on total sales
* Highlighted key revenue contributors

### 3. Profit Analysis

* Analyzed profit by category
* Identified loss-making products

---

## 📈 Key Insights

* 📅 **Seasonality Observed**: Sales peak during certain months (e.g., year-end)
* 🏆 **Top Products Drive Revenue**: A small number of products contribute heavily to total sales
* ⚠️ **Profit ≠ Sales**: Some high-selling products generate low or negative profit
* 📉 **Loss-Making Items Identified**: Requires pricing or cost optimization

---

## 🚀 How to Run the Project

1. Install required libraries:

```bash

```

2. Place the dataset file:

```

```

3. Run the Python script or Jupyter Notebook:

```bash
Sales_Data_Analysis.py
```
